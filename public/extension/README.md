# SARA Chrome Extension

This extension allows SARA to listen for the wake word ("Hey SARA") even when you are on your home screen or another tab.

## How to Install

1. Download this folder to your computer.
2. Open Google Chrome and go to `chrome://extensions/`
3. Turn on **Developer mode** (toggle switch in the top right corner).
4. Click the **Load unpacked** button in the top left.
5. Select this `extension` folder.
6. The extension is now installed! 

## How to Use

1. Make sure your microphone is connected.
2. Say "Hey SARA" from any tab or your home screen.
3. SARA will automatically open in a new tab and start listening!

*(Note: An `icon.svg` is included in this folder for the extension icon).*
